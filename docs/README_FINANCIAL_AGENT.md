# Financial Agent Guide

AI-powered banking assistant for WhatsApp that handles money transfers, balance checks, and transaction history through natural conversations.

## Features

- **Balance Checks**: Check your Paystack balance instantly
- **Money Transfers**: Send money to any Nigerian bank account
- **Transaction History**: View your complete financial activity
- **Account Resolution**: Verify bank account details automatically
- **Recipient Management**: Save frequent recipients for quick transfers
- **Natural Conversations**: Chat in English, Pidgin, or mixed languages

## How It Works

1. **Send a message** via WhatsApp
2. **AI understands** your intent and extracts details
3. **System processes** your request via Paystack API
4. **You get response** in natural, conversational style
